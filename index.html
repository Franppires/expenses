Primeira vez? Configure a nuvem (grátis) em js/firebase-config.js:

  1. Crie um projeto em Firebase Console
  2. Ative Authentication → E-mail/senha
  3. Crie Firestore Database
  4. Copie as chaves do app Web para firebase-config.js
  5. Defina configured: true

As regras de segurança do Firestore estão no arquivo js/cloud.js.
